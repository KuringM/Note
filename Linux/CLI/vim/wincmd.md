`CTRL-W` 窗口控制快捷键
所有的 CTRL-W 命令都可以用`:winc` `:wincmd`执行

### 打开和关闭窗口
|      Command      |        Description        |
|:-----------------:|:-------------------------:|
| `:winc s/S` `:sv` |   horizontal split file   |
|     `:winc v`     |    vertical split file    |
|  `:winc n` `:new` | horizontal creat new file |
|      `:vnew`      |  vertical creat new file  |
|      `:sf **`     |  horizontal split ** file |
|     `:winc q `    |        quit window        |
